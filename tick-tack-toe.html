The game Tic Tac Toe

fun javascript game

I made this two-player game with the use of JavaScript, HTML, and CSS. The longest part of making this game was the JavaScript code, which makes the game interactive.

Live Code

While building this game I learned so much about how to manipulate the DOM, how to use the events in JavaScript to dynamically change what is happening on the page.

The most challenging part was the game logic itself. I also learned that with the use of the browser's local storage I was able to save the player's names which made it useful for future projects.
